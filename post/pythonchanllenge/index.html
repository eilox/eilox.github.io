第1关

http://www.pythonchallenge.com/pc/def/map.html

这关给了个对应关系，K-M，O-Q，E-G，总结规律就是将密文中的每个字母替换成其在字母表中后方的第二个字母。如果是y和z，则回到字母表开头，分别对应a、b。
思路很简单，每个字符转ascii码，将ascii+2再转回字符，就能得到明文。不过里边的空格和标点符号都是明文，需要排除掉。

sentence = " g fmnc wms bgblr rpylqjyrc gr zw fylb. rfyrq ufyr amknsrcpq ypc dmp. bmgle gr gl zw fylb gq glcddgagclr ylb rfyr'q ufw rfgq rcvr gq qm jmle. sqgle qrpgle.kyicrpylq() gq pcamkkclbcb. lmu ynnjw ml rfc spj. "
exceptions =[" ",".",",","'"]

for i in secret:
    if i in exceptions:
        sentence += i
    elif ord(i) > 120 :
        sentence += chr(ord(i)-26+2)
    else:
        sentence += chr(ord(i)+2)
#print(sentence)

另一种可能性，是不能自己给参数赋值，而是根据页面给出的提示进行。那就需要提取提示里的数字。

from urllib.request import urlopen
import re

para = str(16044)
number_fliter = re.compile("\d{4,5}")

for i in range(0,400):
    try:
        ori_url = 'http://www.pythonchallenge.com/pc/def/linkedlist.php?nothing='
        url = ori_url + para
        resp = urlopen(url,timeout=1).read().decode('utf-8')
        para = number_fliter.search(resp).group()
        print(i,para)
    except:
        print('time out')
